PROJECT WEBSITE  

PLATFORM

DESKTOP/MOBILE

SEPT. 2016

I was given the index.html file for this project and was tasked with creating an inviting and friendly website for the app, "diplomacy". Using the colorscheme found in the icon, I was able to create a website that models other app websites. I also tried my hand at UI/UX by creating a mockup of the app, as seen with the image of the iPhone on the webpage. This project was done entirely with HTML and CSS. The responsiveness of the webpage was created myself by using media queries as opposed to using a framework - this gave me complete control over how the elements of the page would interact with varying screen sizes.
